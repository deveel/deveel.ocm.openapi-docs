route

Creates a new Route of Incoming Messages

Authorizations:
oauth2 (create:route)
Request Body schema: application/json
channel
required
string non-empty ^[a-zA-Z0-9_.-]+$

The name of the channel transporting the inbound messages

name
required
string non-empty

A descriptive name of the route (for display)

destinationUrl
required
string non-empty

The HTTP URL address where the inbound message will be delivered

active
boolean

Whether the route must be set as immediately active

retryCount
integer <int32>

The number of retries to deliver the inbound message

secret
string Nullable

A secret code that is used to sign the webhooks

object Nullable

A context that is appended to the message before the delivery (useful for correlation)

Array of objects (RouteMatchRule) Nullable

A list of rules that are used to match a message to the given route

Responses

https://api.ocm.deveel.net/route/
