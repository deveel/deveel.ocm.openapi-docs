Sends a single Message

Authorizations:
oauth2 (send:message)
Request Body schema:
application/json-patch+json
application/json-patch+json
application/json
text/json
application/*+json
required
object (SenderTerminal)

A terminal used to send messages to receivers (typically from a server application)

channel
required
string

The name of the channel that is used to transport the message

object Nullable

Configuration options for modeling the behavior of the channel when transporting the message

object (FallbackMessage)

Represents a message is activated when a previous one failed to be delivered to a receipient

required
object (Terminal)

A terminal allows to send or receive messages through channels

required
object (MessageContent)

Represents the content of a message

object Nullable

A custom-defined context of the message (typically used to correlate).

property name*
any

Responses

https://api.ocm.deveel.net/message/
